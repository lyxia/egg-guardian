# 工资计算和显示相关 Bug

## Bug 1: 实际领取金额与预计金额不符

### 问题描述
在领取今日工资时，预计守护显示的金额与实际增加的余额不一致。

### 重现步骤
1. 打开应用，点击"领取今日工资"按钮
2. 在任务核对弹窗中，选择部分任务完成（例如：完成"认真写完作业"和"早晚刷牙"，未完成"练琴20分钟"和"整理书包"）
3. 观察"预计守护"显示的金额
4. 点击"确认领取"
5. 查看实际增加的余额

### 预期行为
- 预计守护显示：329 蛋币
- 实际应增加：329 蛋币

### 实际行为
- 预计守护显示：329 蛋币
- 实际增加：324 蛋币
- **差额：5 蛋币**

### 测试数据
- 初始余额：3,850
- 领取后余额：4,174
- 实际增加：324（4,174 - 3,850）
- 预期增加：329

### 计算详情
- 基础工资总计：572 蛋币
- 未完成任务扣除：
  - 练琴20分钟：-162
  - 整理书包：-81
  - 扣除合计：-243
- 预期结果：572 - 243 = 329
- 实际结果：324

## Bug 2: 工资条记录信息不完整

### 问题描述
查看工资条记录时，只显示基础工资，没有显示实际领取金额和扣除信息。

### 重现步骤
1. 完成工资领取流程
2. 点击"查看工资条"按钮
3. 查看工资条记录详情

### 预期行为
工资条应该显示：
- 日期：2026年1月14日
- 基础工资：572 蛋币
- 扣除金额：243 蛋币（捣蛋鬼抱走）
- 实际领取：329 蛋币（或实际的 324）

### 实际行为
工资条只显示：
- 日期：2026年1月14日
- 基础工资：572 蛋币

缺少：
- 扣除金额信息
- 实际领取金额

### 影响
用户无法从工资条中了解完整的工资计算过程，不知道哪些任务未完成导致了扣除。
